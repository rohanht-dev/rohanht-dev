## Projects

### [🎬 Automated Video Subtitle Generator](https://github.com/rohanht-dev/automated-video-subtitle-generator)
`Python` `Whisper AI` `FFmpeg` `Streamlit`

End-to-end Python automation pipeline. Takes a video → extracts audio with FFmpeg → transcribes using OpenAI Whisper AI (~90% accuracy) → generates SRT subtitles → embeds back into video. Deployed with a Streamlit interface.

> Built: complete pipeline architecture, Whisper AI integration, SRT timestamp formatting, Streamlit UI

---

### [🎓 Student Management System](https://github.com/rohanht-dev/student_management_system)
`Django` `Python` `MySQL` `Chart.js`

Production-configured Django web app for full student lifecycle management. Features CRUD operations, search, result tracking, attendance, and a Chart.js analytics dashboard.

> Built: MySQL schema design, Django ORM queries, Chart.js dashboard, .env + requirements.txt deployment config

---

### [📦 Inventory Reservation System](https://github.com/rohanht-dev/allo-inventory-reservation)
`Next.js` `TypeScript` `PostgreSQL` `Redis` `Prisma`

Full-stack inventory platform with Redis distributed locking to prevent overselling during concurrent checkouts. Implements reservation expiry handling and stock management workflows.

> Built: Redis locking logic, Prisma ORM with PostgreSQL, reservation expiry system, Next.js API routes

---

### [🛒 E-Commerce Product Store](https://github.com/rohanht-dev)
`HTML5` `CSS3` `JavaScript` `Fetch API`

Responsive multi-page storefront with dynamic product filtering, cart functionality, and Async/Await REST API integration. Cross-device compatible.
