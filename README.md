### 🎬 Automated Video Subtitle Generator

</div>

```
VIDEO INPUT ──► FFmpeg Audio Extraction ──► Whisper AI Transcription ──► SRT Generation ──► Subtitle Embedding
                                                    (~90% accuracy)
```

An end-to-end Python automation pipeline that takes any video file and produces accurately-timed subtitles — no manual work required. The system extracts audio, transcribes using OpenAI's Whisper AI, generates SRT subtitle files, and embeds them back into the video — all accessible via a Streamlit web interface.

### 📦 Inventory Reservation System

</div>

```
Next.js ──► API Routes ──► Redis Locking ──► Prisma ORM ──► PostgreSQL ──► Reservation Expiry
                          (Prevents overselling)
```

A full-stack inventory management platform with distributed locking to prevent overselling during concurrent checkouts. Built with Next.js, PostgreSQL via Prisma ORM, and Redis for reservation locking and expiry workflows.
